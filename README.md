 AI Collision Avoidance Mission Control

Интеллектуальная система прогнозирования и предотвращения столкновений спутников в околоземной орбите.

 Описание проекта

AI Collision Avoidance Mission Control — это программная система поддержки принятия решений для анализа космической обстановки.

Система:

    Выполняет распространение орбит спутников в реальном времени (SGP4)
    Обнаруживает потенциальные опасные сближения
    Рассчитывает гибридный индекс риска столкновения
    Моделирует манёвр уклонения (Δv)
    Визуализирует орбиты в интерактивном 3D-интерфейсе

Проект разработан в рамках соревнования AEROO SPACE AI COMPETITION.

 Использование искусственного интеллекта

    Система использует гибридный подход:
    Физическая модель распространения орбиты (SGP4)
    Анализ минимального расстояния
    Оценка относительной скорости
    Модель гибридного риска (physics + AI-ready архитектура)
    Архитектура готова к интеграции:
    ML-моделей прогнозирования вероятности столкновения
    Нейросетевых алгоритмов оценки рисков
    Автоматической оптимизации манёвров

 Основные функции

    3D-визуализация орбит
    Анимация движения спутников
    Автоматическое определение сближений
    Расчёт расстояния между объектами
    Расчёт индекса риска
    Симуляция манёвра уклонения



 Технологический стек

    Python 3.10+
    VisPy (3D визуализация)
    NumPy
    Matplotlib
    SGP4
    PySide6
    PyInstaller (сборка .app)

 Как запустить проект (из исходного кода)

1️ Клонировать репозиторий

    git clone https://github.com/your_username/ai-collision-avoidance.git
    cd ai-collision-avoidance

2️ Создать виртуальное окружение
    
    python3.10 -m venv venv
    source venv/bin/activate

3️ Установить зависимости

    pip install -r requirements.txt

4️ Запуск приложения

    python -m src.app_desktop.main
